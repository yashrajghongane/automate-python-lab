  8.  Why is eggs a valid variable name while 100 is invalid?
Ans :-
According to Python's Rules for naming variables, a variable name must start with a letter or an underscore(_), and cannot start with number.
Therefore, the eggs is valid name (start with letter ) and 100 is invalid (start with number).
