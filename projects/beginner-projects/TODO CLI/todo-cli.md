# TODO CLI Tool for tracking tasks.
# Version - v0.1
# Features it have :-
``` 
1. Add Taskk 
2. View Task
3. Remove Task 
4. Exit
```
___

# Projects Basic Architecture :-
```
ask for choice - decide the action basis of choice - 
    if choice 1 = input task - store it.
    if choice 2 = show all the tasks.
    if choice 3 = input the task no to delete - delete it.
    if choice 4 = exit the app.
repeat asking continuesly 
```
